# Tool Assignment Project

## Files
- `tool_assignment.py`: Main script to parse input, assign samples to tools, and write output.
- `sample_input.txt`: Sample input file with tools and samples definitions.
- `tool_assignment_output.txt`: Expected output generated by the script.
- `test_tool_assignment.py`: Pytest file to verify output matches expected.

## Usage
1. Ensure Python 3 is installed.
2. Place all files in the same directory.
3. Run the script:
   ```
   python tool_assignment.py
   ```
4. Check `tool_assignment_output.txt` for results.
5. Optionally, run tests:
   ```
   pytest test_tool_assignment.py
   ```
